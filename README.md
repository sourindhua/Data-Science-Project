# Data-Science-Project
Predictive Modelling on a Web Data with a Power BI Dashboard

Developed a predictive modelling project that integrated data scraping, preprocessing, machine learning, and visualization.
Created a Python script to scrape data from a public website, collecting and preparing over 1,000 records for analysis.
Designed an interactive dashboard in Power BI to visualize model predictions and present business-friendly insights.

## 🌐 Data Source
Data was scraped from:
http://books.toscrape.com
